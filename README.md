# Kotlin in a Nutshell
Kotlin ポケットリファレンス（日本語）

## 概要

`TheKotlinProgrammingLanguage.md` 1ファイルにKotlinの主要な構文・機能をコンパクトにまとめた日本語リファレンスです。
